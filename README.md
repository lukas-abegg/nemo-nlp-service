# nemo-nlp-service
Different Prototypes for NLP Services

# Installation
   * you need python 3.7
   * for installation of required python packages run:
 ```
    pip install -r requirements
```

# Run Server
   * To run the server you just need to run
 ```
   python -m flask run
```

# Open Webapp
   * Server runs on port 5000
   * Type into your browser
```
   localhost:5000
```
